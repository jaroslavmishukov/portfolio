# Портфолио Ярослава

Одностраничное портфолио Python-разработчика: бизнес-системы, автоматизация, интеграции, парсинг и Telegram-боты.

## Что внутри

- Главный кейс: система бронирования для базы отдыха на FastAPI, PostgreSQL и Docker.
- Примеры проектов: сервис подписок через чат-бота, агрегатор контента, парсеры и боты для бизнеса.
- Описание услуг: системы бронирования, Telegram-автоматизация, парсинг данных, VPS-деплой, интеграция оплат и доработка существующих проектов.

## Стек

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- Jinja2
- Docker
- nginx
- ЮKassa
- Telegram Bot API
- pytest

## Ключевой опыт

- Разработка сайтов с админ-панелями и базой данных.
- Онлайн-бронирование, статусы заявок, роли сотрудников и отчеты.
- Интеграция платежей, webhooks, возвраты и разделение sandbox/production режимов.
- Telegram-уведомления и боты для бизнес-процессов.
- Деплой на VPS через Docker Compose, nginx и HTTPS.
- Бэкапы, healthcheck, логи и тесты бизнес-логики.
